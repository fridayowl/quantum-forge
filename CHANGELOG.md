## [0.1.0] - 2025-12-30

### Added
- Initial release of QuantumForge
- Circuit optimization with multi-level optimization strategies
- Error mitigation techniques:
  - Zero-Noise Extrapolation (ZNE)
  - Readout error correction
  - Multiple extrapolation methods
- Quantum state analysis tools:
  - Entanglement measures (entropy, concurrence)
  - Fidelity calculations
  - Purity metrics
  - State visualization (Bloch sphere, state city)
- Variational Quantum Eigensolver (VQE):
  - Hardware-efficient ansatz
  - UCCSD ansatz
  - Multiple classical optimizers
- Quantum Approximate Optimization Algorithm (QAOA):
  - Max-Cut solver
  - Customizable layers
  - Approximation ratio calculation
- Noise modeling:
  - Depolarizing noise
  - Thermal relaxation (T1, T2)
  - Amplitude and phase damping
  - Readout errors
  - Kraus operator representations
- Comprehensive examples and documentation
- Unit tests for core functionality
